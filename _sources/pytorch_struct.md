# Pytorch库的结构

- 基础：神经网络支持
    - torch.nn 神经网络支持
    - torch.functional 各种神经网络中常用的函数
    - torch.optim 神经网络优化器
    - torch.autograd 自动求导
    - torch.utils.data 数据集
- Numpy/Scipy
    - torch.linalg
    - torch.fft
    - torch.special
    - torch.sparse
- 函数式编程
    - torch.func
- 分布式与加速
    - torch.distributed
    - torch.amp
